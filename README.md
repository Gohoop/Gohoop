Gohoop
====
Gohoop分布式存储与计算平台(Gohoop Distributed Storage And Compute Platforms)，简称GDSCP。开发语言C++，可运行在Linux平台上。

Purpose
====
1. 平台实现大数据分布式存储。
2. 平台实现任务分布式运算。
3. 平台对外以RPC方式提供服务。
